Katrina's Midterm Project Idea Description

The Idea : Create an interactive website that allows users to play a scavenger hunt style game to find a treasure. The idea is to have the game be quarantine themed, with the online hunt allowing the user to travel to different places around the world virtually. This will be done utilizing images/videos and other interactive elements (aka clues) that drive the user through the website.

These are just a simplified version of wireframes for the different types of pages a user will be able to click through, there will be more interactive elements than shown on the clue screen as of right now, and there will be more images added to other screens.
